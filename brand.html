Wordmark

RawMine is set as a single word, weight 800, with the second half “Mine” in the accent teal. Don’t add a tagline lockup or alter the letter spacing.

Do
  • Keep “Mine” in accent teal on light backgrounds.
  • Use the lighter teal (#5EEAD4) on dark backgrounds.
  • Use the all-ink mono version where color isn’t available.
Don’t
  • Lowercase it (“rawmine”) or split it into two words.
  • Recolor it outside the palette or add effects/shadows.
  • Stretch, condense, or re-space the letters.

Color

A near-black ink on white, one teal accent, and quiet greys for structure. The accent carries all emphasis, so use it sparingly.
